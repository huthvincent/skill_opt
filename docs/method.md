## 第一层：背景——我们到底在优化什么

一个 AI 客服 agent 靠一段可随时增删改的**文字说明书（经验库/skill）**干活。我们的研究：**有真实用户交互日志时，怎么自动把说明书越改越好**——不训练模型、只改文字（training-free）。

前情提要（详见本文档 v2 版历史）：
- 一口气塞 12 条经验 → 实测有害（$18 教训）；
- 改成"一次一条、小范围试药、统计门准入"（v1 六步法）→ 管线冒烟通过；
- **但 v1 有个命门**：准入门靠 LLM 裁判打分，而裁判会被骗。

## 第二层：命门——裁判必被攻破

我们调研了 63 篇文献，结论触目惊心：所有 self-evolve 方法的裁判都在裸奔——要么 agent 自己当裁判（自己批自己的作业），要么裁判冻结不动（被优化器慢慢摸透）。在我们的目标场景（多轮工具对话轨迹），最强裁判配置的判对能力天花板是 AUROC<0.65，跟瞎猜距离不远。

更要命的是**方向性**：优化器不是随机犯错，它会**系统性地朝着裁判的盲区进化**——这就是 reward hacking。裁判是静止的靶子，优化器是不断学习的射手，时间站在射手那边。

## 第三层：Owner 的洞察——那就让裁判也进化（GAN 式对抗）

既然裁判静止必败，就让它跟 agent 一起进化，像 GAN：**做题家**（agent，想通过审核）vs **阅卷官**（裁判，想分辨真做对和假装做对），再加一个 GAN 里没有的显式角色——**职业作弊者（红队）**，专职用固定预算制造能骗过当前阅卷官的假成功。

循环三回合：做题家进化（老的 v1 管线原样保留）→ 作弊者攻击（骗过门的样本进**作弊档案**——妙处：作弊者自己知道自己是假的，档案**天然带标注**，不需要标准答案）→ 阅卷官拿档案给自己打补丁（但补丁要先通过"阅卷官自己的考试"才准上岗）。

调研发现"裁判必须共进化"这个论点 2026 年已成共识（有六七篇在做），**但没有一篇能回答：进化到什么时候算完？** 没有终点的军备竞赛没法部署、没法写论文结论——这个空白就是我们的主贡献。

## 第四层：终点判据——本项目最核心的设计

**GAN 的终点搬过来是错的。** GAN 里"判别器分不清了"=成功（造假像真就是目标）。我们这里"阅卷官挑不出毛病"是**歧义状态**，三种可能从循环内部看一模一样：

1. agent 真的好到无毛病 ✅
2. agent 骗术臻于完美 💀
3. 阅卷官和 agent 一起烂掉——标准被悄悄放水，双方"和解" 💀（文献实锤的共进化失败模式）

**打破歧义靠"笼子"**：一组双方都碰不到的外部锚——冻结的带答案考卷（金锚集）、一份连实验内部都保密的考卷（金丝雀）、以及文字讨好不动摇的硬证据层（工具报错、声称与行为的一致性）。对抗在笼内进行，笼子不参与进化。这就是协议名 **CAGE** 的意象。

**三条件同时成立且持续 k 轮，才算收敛**：

| 条件 | 通俗说法 | 防哪种假收敛 |
|---|---|---|
| C1 真实进步饱和 | 考卷上的真实成绩不再涨 | 还有肉没吃完就收工 |
| C2 作弊产率灭绝 | 职业作弊者拿着固定预算也骗不进来了 | 潜伏漏洞（主动派人试撬门，而不是坐等出事） |
| C3 锚上精度不降 | 阅卷官做老考卷的成绩没有退步 | 双方一起放水 |

**单值指标——欺骗溢价（DP）**：骗过门一次要花的钱 ÷ 真做对一次要花的钱。
- 现有所有方法默默活在 DP≪1（作弊便宜到离谱）；
- **DP 持续 >1 = 作弊比做真更贵 = 理性的优化器只剩"真做对"一条路 = 终点。**

一句话：**GAN 收敛于"以假乱真"，CAGE 收敛于"作假不如做真"。** 调研确认这个判据没人用过，且能形式化挂靠博弈论的 exploitability（自博弈领域的标准收敛概念，我们把它预算化、施加在裁判身上）。

## 第五层：为什么这个组合能守住（撞车地图速览）

- "对抗共进化"框架：被占（Prover-Verifier Games、EvoRubrics、ARCO、Red Queen Gödel Machine、CoEvoSkills）——但它们**全是 RL 权重训练**（除 CoEvoSkills）且**全体没有终止判据**（RQGM 明说没有）；
- 我们守的四点组合（一手核实无人占）：**锚定终止证书** ×**欺骗溢价判据** × **training-free** × **多轮工具轨迹域**；
- 命名策略：标题主打证书（*Cheating Must Not Pay* 方向），GAN 类比降为 intro 一句话——因为"GAN+agent"的字面组合已被 MAGAIL/GAR/LLM-GAN 占浑，且框架是弱卖点、证书是强卖点。
