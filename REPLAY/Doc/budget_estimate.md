## 3. 压成本杠杆（按优先级）

1. **prompt caching**：τ²-bench 每集重复注入同一份 policy 文档（约 3–5k tok），Anthropic cache 命中价 1/10——预计省 30–50% 输入费。E0.2 时验证 litellm 参数透传。
2. **测试集抽样**：114→50 任务、pass^2 代替 pass^4（表格已按此假设）。
3. **主 agent 用 haiku**：方法贡献与 agent 强弱正交，主表可用 haiku 出全量数字 + sonnet 只跑 1 域做泛化性验证（混合方案 ≈ **$1,800–2,200**，推荐）。
4. Bedrock 与直连同价，无套利空间；换机迁移不影响本估算。
